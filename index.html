¿Qué es un Grafo? Tipos de Grafos Representación de Grafos

– ¿Qué es un Grafo?

Un grafo es una estructura de datos compuesta por nodos (llamados vértices) y conexiones entre ellos (llamadas aristas). Los grafos permiten modelar relaciones entre entidades, por lo que se utilizan en redes sociales, mapas, análisis de rutas, inteligencia artificial, compiladores y más.

Matemáticamente, un grafo G se define como:
G = (V, E) donde V es el conjunto de vértices y E es el conjunto de aristas.

– Tipos de Grafos

Los grafos pueden clasificarse de diferentes maneras dependiendo de sus características:

  • Grafo dirigido: Sus aristas tienen dirección.
  • Grafo no dirigido: Las aristas no tienen dirección.
  • Grafo ponderado: Cada arista tiene un valor o peso.
  • Grafo no ponderado: Las aristas no tienen pesos.
  • Grafo completo: Todos los nodos están conectados entre sí.
  • Grafo bipartito: Sus nodos pueden dividirse en dos grupos.

– Representación de Grafos (Listas, Matrices y Más)

Para implementar grafos en programación, es necesario elegir una representación eficiente según el tipo de operaciones que se realizarán. Las tres representaciones más comunes son:

1. Lista de Adyacencia

Representa el grafo usando una lista o arreglo donde cada índice almacena los nodos adyacentes. Es ideal para grafos dispersos. 

Grafo:
1 → 2, 3
2 → 4
3 → 1
4 → (sin conexiones)

2. Matriz de Adyacencia

Es una matriz NxN donde cada posición indica si existe una arista entre los vértices. 

    1  2  3  4
1 [0, 1, 1, 0]
2 [0, 0, 0, 1]
3 [1, 0, 0, 0]
4 [0, 0, 0, 0]

3. Lista de Aristas

Se almacena una lista con todas las aristas del grafo. Es útil para algoritmos como Kruskal. 

(1,2)
(1,3)
(2,4)
(3,1)

Cada representación tiene ventajas distintas: ✔ La matriz es rápida para consultar conexiones. ✔ La lista de adyacencia ahorra memoria. ✔ La lista de aristas sirve para algoritmos de árboles de expansión mínima.
